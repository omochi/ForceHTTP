# ForceHTTP

Handmade HTTP library build on the *4th* network library, Network.framework.

# Features

## supports

- HTTPS
- IPv6 routes
- bypassing ATS on iOS
- connection pooling
- redirection
- x-www-form-urlencoded

## not support yet

- on disk download
- pipelining
- mime
